# Uso del Generador de Campañas

1. Abre la pestaña GitHub Models en tu repositorio.
2. Modifica los archivos YAML en la carpeta `prompts` para personalizar tus campañas.
3. Ejecuta los prompts con diferentes modelos para generar contenido.
4. Usa los ejemplos en `examples/` para inspirarte y validar.
5. Ejecuta las pruebas en `tests/` para asegurar la calidad de los prompts.

Puedes usar parámetros para ajustar la creatividad (`temperature`), la extensión (`max_tokens`) y más.

## Recursos
